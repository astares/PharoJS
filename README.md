## Install

To install PharoJS, perform the code appropriate to your Pharo version in a playground:

### Pharo 9 and Pharo 10

```smalltalk
Metacello new
  baseline: 'PharoJS';
  repository: 'github://PharoJS/PharoJS:pharo9';
  load
```

### Pharo 7 and Pharo 8

```smalltalk
Metacello new
  baseline: 'PharoJS';
  repository: 'github://PharoJS/PharoJS:pharo8';
  load
```
